## What is Meta Content System?

Meta Content System is the portable content pipeline behind Dev-Op-Typer (Windows) and linux-dev-typer (cross-platform). It ingests source code files, normalizes them deterministically, computes difficulty metrics, and produces an indexed library that both apps consume identically regardless of platform.

Zero external dependencies. Pure .NET 8 library built entirely on the BCL.
